# THU_INSC_Course_Guide

清华大学网络研究院课程攻略开源共享项目

本仓库持续收录我在读研期间所修课程及完成的大作业项目，现已全部开源共享。希望能为各位同样“苦逼”的小伙伴们提供一些参考，让大家在繁忙的课程与科研之余，也能有机会摸摸鱼、喘口气~

此外，我也会陆续在这里记录自己在读研期间踩过的各种坑，以及一些实用的经验和心得，希望能对大家有所帮助。欢迎持续关注，一起交流进步。

本项目应该是网络研究院建院以来，首个成体系的课程开源共享仓库。由于个人水平有限，难免存在疏漏或不完善之处，欢迎大家批评指正，多提宝贵意见！
